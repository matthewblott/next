Get command Id of action:
Code -> Settings -> Keyboard Shortcuts (Command + ,)
Press Record Keys and watch the Recording Keys symbol appear
Press the keys you want to find the Id for
Press Record Keys again to stop and watch the Recording keys symbol disappear
Right-click and press "Copy Command ID"

Set keybindings here:
Code -> Settings -> Profile -> Default -> Show Profile Contents
Keyboard Shortcuts -> settings.json

User settings:
Code -> Settings -> Profile -> Default -> Show Profile Contents
Settings -> settings.json
